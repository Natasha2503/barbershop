1. Вводные данные:
Расписание работы парикмахерской: по будням с 10:00 до 20:00 мск
Рабочий день парикмахерской разбит на часовые интервалы, один час - это одна запись.
В парикмахерской есть два мастера: Алиса и Боб.
Пользователь открывает веб-приложение, переходя по ссылке с постоянным уникальным идентификатором пользователя в URL: mytesttaskhost.com/haircut?user-id=12345
Мастер не может одновременно стричь двух человек.
Для записи человека нужно зафиксировать его имя и номер телефона.

Требования:
Нужно разработать web-интерфейс, который позволит пользователю записаться на стрижку.
Нужно разработать web-интерфейс, который показывает парикмахерской все текущие записи.

Технологии:
Решение должно представлять из себя web-приложение, запущенное на localhost
Выбор web-фреймворка - на своё усмотрение
В качестве хранилища использовать PostgreSQL
В проекте должны быть реализованы unit-тесты
Тестовое задание должно быть оформлено в виде проекта на github
